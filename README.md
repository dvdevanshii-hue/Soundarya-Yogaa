# 🧘‍♀️ Soundarya Yoga – AI Powered Face Yoga Platform

## 📌 Overview
Soundarya Yoga is an AI-powered Face Yoga platform designed to help users improve facial wellness naturally through personalized face yoga exercises. The website analyzes facial concerns using face scanning technology and recommends suitable yoga routines, skincare tips, and wellness guidance.

This project was developed as a group project with the aim of combining wellness and technology to create an interactive and user-friendly experience.

## 🚀 How It Works

1. User registers/logs into the platform.
2. User uploads or scans their face.
3. AI analyzes facial features and concerns.
4. Personalized face yoga exercises are recommended.
5. Users can follow routines and wellness tips.
6. Chatbot provides additional guidance and support.
